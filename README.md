# DmrMovie

## Proje Hakkında

DmrMovie, film ve dizi önerileri, incelemeleri veya izleme listeleri oluşturmak amacıyla geliştirilen modern bir web uygulamasıdır. Kullanıcı dostu arayüzü sayesinde kolayca film/dizi araması yapabilir, popüler yapımları keşfedebilir ve favori listenizi düzenleyebilirsiniz.

## Kullanılan Teknolojiler ve Diller

- **TypeScript**: Uygulama mantığı ve tip güvenliği için.
- **JavaScript**: Web dinamikleri ve etkileşim için.
- **CSS**: Tasarım ve stil düzenlemeleri için.

## Kullanım

1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/AnonymousHer/DmrMovie.git
   ```
2. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı başlatın:
   ```bash
   npm start
   ```
